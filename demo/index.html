Generate Documents

Click Legalese / Generate Documents.

Legalese will create a Legalese Root folder in your Google Drive. Legalese will output under that folder.

Under Legalese Root you will find an output directory timestamped with the current time.

Under that folder you will see a bunch of Google Docs containing sample output.
